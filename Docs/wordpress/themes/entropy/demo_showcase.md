#### RokFeatureTable

The **RokFeatureTable** widget makes it easy to create a table to compare multiple products, pricing plans, and more. You will find the options used in this widget below.

| Option           | Setting |  
| :--------------- | :------ |  
| Title            | Blank   |  
| Highlight Column | 1       |  
| Built-in CSS     | No      |
