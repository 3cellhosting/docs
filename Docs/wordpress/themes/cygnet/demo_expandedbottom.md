Here is a breakdown of options changes you will want to make to match the demo.

| Option            | Setting                          |
| :---------------- | :---------                       |
| Title             | `Extendability`                  |
| Custom Variations | `fp-expandedbottom rt-big-title` |

Leaving everything else at its default setting, select **Save**.
