#### Gantry Login Form

The login form located in this area of the page is actually a **Gantry Login Form** widget. Here are the widget options you will need to change in order to match the demo.

| Option            | Setting    | 
| :---------------- | :--------- | 
| Title             | Login Form | 
| User Greeting     | Hi,        | 
| Pre-text          | Blank      | 
| Post-text         | Blank      | 
| Custom Variations |            |
