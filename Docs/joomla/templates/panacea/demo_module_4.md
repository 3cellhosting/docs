Log me in
-----

![][demo]

This area of the front page is a **Login** module. You will find the settings used in our demo below.

### Details

![][demo2]

| Option     | Setting               |
| :--------- | :-------------------- |
| Title      | `Log me in`           |
| Show Title | Show                  |
| Position   | sidebar-a             |
| Status     | Published             |
| Access     | Public                |
