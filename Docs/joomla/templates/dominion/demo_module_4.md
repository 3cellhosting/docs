Member Login

This area of the front page is a standard **Login** module. Here are the settings we used in our demo.

### Details

![][demo2]

| Option      | Setting        |
| :---------- | :----------    |
| Title       | `Member Login` |
| Show Title  | Show           |
| Position    | sidebar-a      |
| Status      | Published      |
| Access      | Public         |
