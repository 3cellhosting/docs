Main Menu
-----

![][demo]

The **Main Menu** module is a standard **Menu** module. Here is a breakdown of the options selected to make it appear as it does on the front page of our demo.

### Details

![][demo2]

| Option      | Setting     |
| :---------- | :---------- |
| Title       | `Main Menu` |
| Show Title  | Show        |
| Position    | sidebar-b   |
| Status      | Published   |
| Access      | Public      |

### Basic Options

![][demo3]

| Option              | Setting   |  
| :------------------ | :-------- |  
| Select Menu         | Main Menu |  
| Limit Levels        | No        |  
| Start Level         | 0         |  
| End Level           | 0         |  
| Show Sub-menu Items | No        |  
| Themes              | Default   |  

### Advanced Options

![][demo4]

| Option              | Setting     |
| :----------         | :---------- |
| Module Class Suffix | `square4`   |
