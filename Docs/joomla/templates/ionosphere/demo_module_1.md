Top Menu
-----
![][demo]

:   1. **Menu** [15%, 46%, se]

The menu located at the top of the demo site is a standard **Menu** module. Here is a breakdown of the options selected to make it appear as it does on the front page of our demo.

### Details
![][demo2]

| Option            | Setting           |  
| :---------------- | :---------------- |  
| Title             | `Top Menu`        |  
| Show Title        | Hide              |  
| Position          | header-b          |  
| Status            | Published         |  
| Access            | Public            |   
| Language          | All               |  
| Note              | Blank             |  

### Basic Options
![][demo3]

| Option              | Setting |  
| :------------------ | :------ |  
| Start Level         | 1       |  
| End Level           | 1       |  
| Show Sub-menu Items | No      |  

### Advanced Options
![][demo4]

| Option              | Setting   |  
| :------------------ | :-------- |  
| Module Class Suffix | `topmenu` |
