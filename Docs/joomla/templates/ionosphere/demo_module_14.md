FP Bottom Menu
----
![][demo]

This area of the site is a **mod_menu** module. A Module Class Suffix is used to make its look match the theme of the Ionosphere template.

### Details
![][demo2]

| Option            | Setting           |  
| :---------------- | :---------------- |  
| Title             | `More Features`   |  
| Show Title        | Hide              |  
| Position          | sidebar-a         |  
| Status            | Published         |  
| Access            | Public            |   
| Language          | All               |  
| Note              | Blank             |  

### Basic Options
![][demo3]

| Option              | Setting          |  
| :------------------ | :--------------- |  
| Start Level         | 1                |  
| End Level           | All              |  
| Show Sub-menu Items | No               |  

### Advanced Options
![][demo4]

| Option              | Setting                 |  
| :------------------ | :---------------------- |  
| Module Class Suffix | `horizmenu bottom-menu` |
