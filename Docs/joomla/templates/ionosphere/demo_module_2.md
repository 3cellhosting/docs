FP Feature
-----
![][demo]
The RokSprocket Showcase module used near the top of the front page is a great way to feature some of your site's more notable articles or areas of interest.

### Details
![][demo2]

| Option           | Setting      |  
| :--------------- | :----------- |  
| Title            | `FP Feature` |  
| Show Title       | Hide         |  
| Access           | Public       |  
| Position         | showcase-a   |  
| Status           | Published    |  
| Content Provider | Joomla       |  
| Type             | Features     |
