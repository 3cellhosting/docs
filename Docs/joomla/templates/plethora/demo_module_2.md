Top Menu
-----

![][demo]

:   1. **menu** [10%, 70%, se]

This area of the front page is a **menu** module. You will find the settings used in our demo below.

### Details

![][demo2]

| Option     | Setting         |
| :--------- | :-------------- |
| Show Title | Hide            |
| Position   | top-b           |
| Status     | Published       |
| Access     | Public          |
| Language   | All             |
| Note       | Blank           |

### Module

![][demo3]

| Option              | Setting     |
| :------------------ | :---------- |
| Title               | `Top Menu`  |
| Select Menu         | Top Menu    |
| Base Item           | Current     |
| Start Level         | 1           |
| End Level           | 1           |
| Show Sub-menu Items | No          |

### Advanced

![][demo4]

| Option              | Setting                   |
| :------------------ | :-----------------------  |
| Module Class Suffix | `rt-horizmenu fp-topmenu` |
