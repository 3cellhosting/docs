Top News
-----

![][demo]

We used a **RokSprocket** module with the **Lists** layout to make up this area of the front page. You will find the settings used in our demo below.

### Details

![][demo2]

| Option           | Setting     |
| :----------      | :---------- |
| Title            | `Top News`  |
| Show Title       | Show        |
| Access           | Public      |
| Position         | maintop-a   |
| Status           | Published   |
| Content Provider | Joomla!     |
| Type             | Lists       |
