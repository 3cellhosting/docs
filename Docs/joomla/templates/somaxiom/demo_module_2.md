Header Menu
-----

![][demo]

The menu area of the sidebar is a **RokNavMenu** module. Here is a breakdown of the options selected to make it appear as it does on the front page of our demo.

### Details

![][demo2]

|   Option   |    Setting    |
| :--------- | :------------ |
| Title      | `Header Menu` |
| Show Title | Hide          |
| Position   | header-c      |
| Status     | Published     |
| Access     | Public        |

### Options

![][demo3]

|        Option       |   Setting   |
| :------------------ | :---------- |
| Select Menu         | Header Menu |
| Limit Levels        | Yes         |
| Start Level         | 0           |
| End Level           | 0           |
| Show Sub-menu Items | No          |
| Themes              | Default     |

### Advanced Options

![][demo4]

|        Option       | Setting  |
| :------------------ | :------- |
| Module Class Suffix | `number` |
