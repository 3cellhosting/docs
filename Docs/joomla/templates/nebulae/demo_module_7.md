Photo Preview
-----

![][demo]

This area of the front page is a **RokGallery** module. Here are the settings we used to make it look the way it does in the demo.

### Details

![][demo2]

| Option     | Setting                  |  
| :--------- | :----------------------- |  
| Title      | `Photo Preview`          |  
| Show Title | Show                     |  
| Access     | Public                   |  
| Position   | footer-c                 |  
| Status     | Published                |  

### Basic Options

![][demo3]

| Option              | Setting                     |  
| :------------------ | :-------------------------- |  
| Gallery             | Photo                       |  
| Link Type           | Link to RokBox1 Slice Image |  
| Default Linked Item | - Home                      |  
| Show Title          | No                          |  
| Show Caption        | No                          |  
| Sort By             | Updated                     |  
| Sort Direction      | Descending                  |  
| Slice Limit         | 6                           |  
| Gallery Style       | Dark                        |  
| Gallery Layout      | Grid                        |  
| Grid Columns        | 3                           |  
| Background Image    | Blank                       |  

### Advanced Options

![][demo4]

| Option              | Setting                                                           |  
| :------------------ | :---------------------------------------------------------------- |  
| Module Class Suffix | `nopaddingright nopaddingbottom nomarginbottom fp-footer-gallery` |
