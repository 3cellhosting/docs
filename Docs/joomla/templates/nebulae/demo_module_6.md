RokGallery Features
-----

![][demo]

This area of the front page is a **mod_roknavmenu** module. You will find the settings used in our demo below.

### Details

![][demo2]

| Option     | Setting                     |  
| :--------- | :-------------------------- |  
| Title      | `RokGallery Features`       |  
| Show Title | Show                        |  
| Position   | footer-b                    |  
| Status     | Published                   |  
| Access     | Public                      |  

### Basic

![][demo3]

| Option              | Setting       |  
| :------------------ | :------------ |  
| Select Menu         | RokGallery    |  
| Start Level         | 1             |  
| End Level           | All           |  
| Show Sub-menu Items | No            |
