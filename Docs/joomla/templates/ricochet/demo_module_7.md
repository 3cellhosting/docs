Must See
-----

![](assets/demo_7.jpeg)

We used a **RokSprocket** module with the **Strips** layout to make up this area of the front page. You will find the settings used in our demo below.

### Details

![](assets/demo_7a.jpeg)

|      Option      |   Setting   |
| :--------------- | :---------- |
| Title            | `Must See`  |
| Show Title       | Show        |
| Access           | Public      |
| Position         | extension-a |
| Status           | Published   |
| Content Provider | Simple      |
| Type             | Strips      |

### Simple Item Example

| Option |                                      Setting                                      |
