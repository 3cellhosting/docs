Login Form
-----

![][demo]

This area of the front page is a standard **Login** module. 

### Details

![][demo2]

| Option     | Setting         |  
| :--------- | :-------------- |  
| Title      | `Login Form`    |  
| Show Title | Show            |  
| Position   | sidepanel       |  
| Status     | Published       |  
| Access     | Public          |  

### Basic Options

![][demo3]

| Option                  | Setting   |  
| :---------------------- | :-------- |  
| Pre-Text                | Blank     |  
| Post-Text               | Blank     |  
| Login Redirection Page  | Default   |  
| Logout Redirection Page | Default   |  
| Show Greeting           | Yes       |  
| Show Name/Username      | User Name |  
| Encrypt Login form      | No        |  

### Advanced Options

| Option              | Setting  |  
| :------------------ | :------- |  
| Module Class Suffix | `title3` |
