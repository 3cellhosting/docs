Main Menu (Sidebar)
-----

![][demo]

The menu area of the sidebar is a standard **Menu** module. Here is a breakdown of the options selected to make it appear as it does on the front page of our demo.

### Details

![][demo2]

| Option     | Setting     |  
| :--------- | :---------- |  
| Title      | `Main Menu` |  
| Show Title | Show        |  
| Position   | sidebar-a   |  
| Status     | Published   |  
| Access     | Public      |  
| Language   | All         |  
| Note       | Blank       |  

### Basic Options

![][demo3]

| Option              | Setting   |  
| :------------------ | :-------- |  
| Select Menu         | Main Menu |  
| Start Level         | 1         |  
| End Level           | All       |  
| Show Sub-menu Items | No        |  

### Advanced Options

![][demo4]

| Option              | Setting |  
| :------------------ | :------ |  
| Module Class Suffix | `_menu` |
