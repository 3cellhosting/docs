Photo Gallery
-----

![][demo]

This area of the front page is a **RokGallery** module. Here are the settings we used to make it look the way it does in the demo.

### Details

![][demo2]

| Option     | Setting                  |  
| :--------- | :----------------------- |  
| Title      | `Photo Gallery`          |  
| Show Title | Show                     |  
| Access     | Public                   |  
| Position   | footer-a                 |  
| Status     | Published                |  

### Basic Options

![][demo3]

| Option                    | Setting                      |  
| :------------------------ | :--------------------------- |  
| Link Type                 | Link to RokBox 1 Slice Image |  
| Default Linked Item       | - RokGallery                 |  
| Show Title                | No                           |  
| Show Caption              | No                           |  
| Sort By                   | Title                        |  
| Sort Direction            | Ascending                    |  
| Slice Limit               | 9                            |  
| Gallery Style             | Light                        |  
| Gallery Layout            | Grid                         |  
| Grid Columns              | 3                            |  
| Select a Background-Image | Blank                        |
