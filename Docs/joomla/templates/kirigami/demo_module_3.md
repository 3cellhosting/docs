FP MainTop
-----
![][demo]
We used a **RokSprocket** module with the **Mosaic** layout to make up this area of the front page. You will find the settings used in our demo below.

### Details
![][demo2]

| Option           | Setting                           |  
| :--------------- | :-------------------------------- |  
| Title            | `FP MainTop - RokSprocket Mosaic` |  
| Show Title       | Hide                              |  
| Access           | Public                            |  
| Position         | maintop-a                         |  
| Status           | Published                         |  
| Content Provider | Joomla                            |  
| Type             | Mosaic                            |
