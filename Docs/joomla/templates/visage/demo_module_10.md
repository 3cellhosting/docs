Site Map
-----

![][demo]

:   1. **menu** [18%, 6%, se]

This area of the front page is a standard **Menu** module. Here is a breakdown of the options selected to make it appear as it does on the front page of our demo.

### Details

![][demo2]

| Option     | Setting    |  
| :--------- | :--------- |  
| Title      | `Site Map` |  
| Show Title | Show       |  
| Position   | footer-a   |  
| Status     | Published  |  
| Access     | Public     |  

### Basic Options

![][demo3]

| Option              | Setting  |  
| :------------------ | :------- |  
| Select Menu         | Bottom Menu |  
| Start Level         | 1        |  
| End Level           | All      |  
| Show Sub-menu Items | No       |  

### Advanced Options

![][demo4]

| Option              | Setting           |  
| :------------------ | :---------------- |  
| Module Class Suffix | `footermenu` |
