RokAjaxSearch
-----

![][demo]

This area of the front page is a **RokAjaxSearch** module. You will find the settings used in our demo below.

### Details

![][demo2]

| Option     | Setting            |  
| :--------- | :----------------- |  
| Title      | `RokAjaxSearch`    |  
| Show Title | Hide               |  
| Position   | utility-c          |  
| Status     | Published          |  
| Access     | Public             |
