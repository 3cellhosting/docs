More Features
----
![][demo]

:   1. **Menu** [7%, 19%, se]

The menu located in the sidebar of the demo site is a standard **Menu** module. Here is a breakdown of the options selected to make it appear as it does on the front page of our demo.

### Details
![][demo2]

| Option            | Setting           |  
| :---------------- | :---------------- |  
| Title             | `More Features`   |  
| Show Title        | Hide              |  
| Position          | sidebar-a         |  
| Status            | Published         |  
| Access            | Public            |   
| Language          | All               |  
| Note              | Blank             |  

### Basic Options
![][demo3]

| Option              | Setting          |  
| :------------------ | :--------------- |  
| Start Level         | 1                |  
| End Level           | All              |  
| Show Sub-menu Items | No               |
