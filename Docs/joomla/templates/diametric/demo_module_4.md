FP Feature A
-----
![][demo]

This area of the front page is a **mod_custom** module. You'll find the settings used in our demo below.

### Details
![][demo2]

| Option     | Setting        |  
| :--------- | :------------- |  
| Title      | `FP Feature A` |  
| Show Title | Hide           |  
| Position   | feature-a      |  
| Status     | Published      |  
| Access     | Public         |  
| Language   | All            |  
| Note       | Blank          |
