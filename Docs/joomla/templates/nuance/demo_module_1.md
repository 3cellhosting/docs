Breadcrumbs
-----

![][demo]

The Site Search area of the page is a **Breadcrumbs** module that allows visitors to see where they are in relation to the home page of your site.

### Details

![][demo2]

| Option      | Setting       |
| :---------- | :----------   |
| Title       | `Breadcrumbs` |
| Show Title  | Hide          |
| Position    | top-a         |
| Status      | Published     |
| Access      | Public        |

### Module

![][demo3]

| Option              | Setting       |
| :----------         | :-----------  |
| Show You Are Here   | No            |
| Show Home           | Yes           |
| Text for Home Entry |               |
| Show Last           | Yes           |
| Text Separator      |               |
