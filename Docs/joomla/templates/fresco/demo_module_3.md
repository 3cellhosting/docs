FP Showcase
-----
![][demo]
The RokSprocket Showcase module used near the top of the front page is a great way to feature some of your site's more notable articles or areas of interest.

### Details
![][demo2]

| Option           | Setting                                |  
| :--------------- | :------------------------------------- |  
| Title            | `FP Showcase A - RokSprocket Features` |  
| Show Title       | Hide                                   |  
| Access           | Public                                 |  
| Position         | showcase-a                             |  
| Status           | Published                              |  
| Content Provider | Joomla                                 |  
| Type             | Features                               |  

### Layout Options
![][demo3]

| Option           | Setting   |  
| :--------------- | :-------- |  
| Display Limit    | ∞         |  
| Theme            | Showcase  |  
| Article Titles   | Show      |  
| Article Text     | Show      |  
| Preview Length   | ∞         |  
| Strip HTML Tags  | Yes       |  
| Arrow Navigation | Show      |  
| Pagination       | Show      |  
| Animation        | Crossfade |  
| Autoplay         | Disable   |  
| Autoplay Delay   | 5         |  
| Image Resize     | Disable   |
