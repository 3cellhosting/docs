RokAjaxSearch
-----

![][demo]

:   1. **RokAjaxSearch** [37%, 52%, se]

The Site Search area of the front page is a **mod_rokajaxsearch** module that allows visitors to search your site using the powerful RokAjaxSearch tool.

### Details
![][demo2]

| Option            | Setting            |  
| :---------------- | :----------------- |  
| Title             | `RokAjaxSearch`    |  
| Show Title        | Hide               |  
| Position          | header-b           |  
| Status            | Published          |  
| Access            | Public             |
