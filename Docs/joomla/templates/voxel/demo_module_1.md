Utility Menu
-----
![][demo]

:   1. **Menu** [46%, 8%, se]

The menu located in the sidebar of the demo site is a standard **Menu** module. Here is a breakdown of the options selected to make it appear as it does on the front page of our demo.

### Details
![][demo2]

| Option            | Setting           |  
| :---------------- | :---------------- |  
| Title             | `Utility Menu`    |  
| Show Title        | Hide              |  
| Position          | utility-a         |  
| Status            | Published         |  
| Access            | Public            |   
| Language          | All               |  
| Note              | Blank             |  

### Basic Options
![][demo3]

| Option              | Setting          |  
| :------------------ | :--------------- |  
| Start Level         | 1                |  
| End Level           | All              |  
| Show Sub-menu Items | No               |  

### Advanced Options
![][demo4]

| Option              | Setting                                              |  
| :------------------ | :--------------------------------------------------- |  
| Module Class Suffix | `horizmenu nomarginall nopaddingtop nopaddingbottom` |
