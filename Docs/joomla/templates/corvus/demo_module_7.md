Latest News
-----
![][demo]
We used a **RokSprocket** module with the **Tabs** layout to make up this area of the front page. You'll find the settings used in our demo below.

### Details
![][demo2]

| Option           | Setting       |  
| :--------------- | :------------ |  
| Title            | `Latest News` |  
| Show Title       | Show          |  
| Access           | Public        |  
| Position         | sidebar-b     |  
| Status           | Published     |  
| Content Provider | Simple        |  
| Type             | Tabs          |
