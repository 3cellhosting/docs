Popular Features
-----
![][demo]
We used a **RokSprocket** module with the **Strips** layout to make up this area of the front page. You'll find the settings used in our demo below.

### Details
![][demo2]

| Option           | Setting            |  
| :--------------- | :----------------- |  
| Title            | `Popular Features` |  
| Show Title       | Show               |  
| Access           | Public             |  
| Position         | mainbottom-b       |  
| Status           | Published          |  
| Content Provider | Joomla             |  
| Type             | Strips             |  

### Layout Options
![][demo3]

| Option            | Setting              |  
| :---------------- | :------------------- |  
| Theme             | Default              |  
| Display Limit     | 6                    |  
| Preview Length    | 50                   |  
| Strip HTML Tags   | No                   |  
| Article Details   | Show Author and Date |  
| Previews Per Page | 3                    |  
| Items Per Row     | 3                    |  
| Arrow Navigation  | Show                 |  
| Pagination        | Hide                 |  
| Animation         | Scale Out            |  
| Autoplay          | Disable              |  
| Autoplay Delay    | 5                    |  
| Image Resize      | Disable              |
